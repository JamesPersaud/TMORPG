TMORPG is a multiplayer role playing game set in a tiny world.

Right, that's the theme done then.

Navigate the world with W,A,S,D or the ARROW KEYS
Use the SPACE bar to attack monsters.

Joining the game will place you in the village but the real action is in the dungeon below. There is no way to win the game, but who wants to live forever when you can become a legend?

Please do give yourself a name or else you'll end up with a random number :(

Good luck!
